This is a list of sessions from the event, based on the schedule in the XML files.
 
----------------------------------------------------------------------------------- 
 
## Title: Temporal table e Query Store - Extraindo o maximo de vantagem do SQL Server
 
**Abstract**:
Nesta sessão vamos olhar para duas features do SQL Server 2016 o Temporal Table e Query Store e vamos avaliar como obter o Maximo de vantagem dessas duas excelentes features.
 
**Speaker(s):**
- Marcelo Fernandes
 
*Track and Room*: Database Development - Sala
 
----------------------------------------------------------------------------------- 
 
 
## Title: Remote Desktop Services no Windows Azure - Deep Dive
 
**Abstract**:
Já conhece o Remote Desktop Services no Azure? Não? Ok, então vamos abordar essa grande “feature” do Azure do qual podemos virtualizar aplicações através da nuvem sem a necessidade de ter uma infraestrutura física, mas se você já conhece, não se preocupe, pois estaremos abordando novos recursos “in place” e também iremos mostrar como criar um ambiente completo de RDS com alta disponibilidade utilizando o SQL com apenas alguns cliques. De modo dinâmico, apresentaremos conteúdos exclusivos e também queremos ouvir pensamentos não filtrados sobre virtualização. Você não vai querer perder essa sessão!
 
**Speaker(s):**
- Rafael Felipe
 
*Track and Room*: Database Administration - Auditorio
 
----------------------------------------------------------------------------------- 
 
 
## Title: SQL Server vNext
 
**Abstract**:
Um overview sobre o SQL Server 2017. Fique ligado no que está por vir.
 
**Speaker(s):**
- Gustavo Moura Fé
 
*Track and Room*: Database Administration - Auditorio
 
----------------------------------------------------------------------------------- 
 
 
## Title: Criando modelos preditivos com Azure Machine Learning
 
**Abstract**:
Machine Learning é o assunto do momento e faz parte da profissão mais cobiçada destes últimos tempos. Aproveite esta sessão com MVP em Data Platform, Diego Nogare, na qual será apresentado como criar um modelo preditivo de ponta a ponta, desde o momento da aquisição do dado até a interação com aplicativos externos consumindo o processo de predição através de WebServices com REST API.
 
**Speaker(s):**
- Diego Nogare
 
*Track and Room*: BI  Big Data - Anfiteatro
 
----------------------------------------------------------------------------------- 
 
 
## Title: Indices para Iniciantes - Clustered, Nonclustered,  Scan, Seek, Lookup e Muito Mais
 
**Abstract**:
Qual a diferença entre um Índice Clustered e um Índice Nonclustered? O que é um Seek, Scan, Lookup,  Include e Fillfactor?

Veja isso e muito mais nessa palestra.
 
**Speaker(s):**
- Fabrício Lima
 
*Track and Room*: Database Development - Sala
 
----------------------------------------------------------------------------------- 
 
 
## Title: High Availability on SQL SERVER and ORACLE
 
**Abstract**:
Quer conhecer as principais soluções de alta disponibilidade do SQL SERVER e do ORACLE ?
Esta é a seção para você! Em uma hora demonstraremos as principais soluções de alta disponibilidade e de recuperação de desastres do SQL SERVER (Always On Failover Cluster Instances e Availability Groups) e do ORACLE (DataGuard e RAC), comparando suas semelhanças e diferenças.
 
**Speaker(s):**
- Marcelo Adade
 
*Track and Room*: Database Administration - Auditorio
 
----------------------------------------------------------------------------------- 
 
 
## Title: Columnstore Indexes - Como ele pode melhorar meu data warehouse?
 
**Abstract**:
Nesta sessão abordaremos o funcionamento dos índices columnstore no SQL Server e como ele pode melhorar a performance do nosso Data Warehouse. Quando ele é útil, quando não é, como manter... entre outros...
 
**Speaker(s):**
- Bruno Feldman
 
*Track and Room*: BI  Big Data - Anfiteatro
 
----------------------------------------------------------------------------------- 
 
 
## Title: 5 dicas de performance tuning para desenvolvedores
 
**Abstract**:
Nessa sessão iremos mostrar 5 dicas de performance tuning para desenvolvedores, mostraremos exemplos práticos do dia-a-dia, após essa sessão o desenvolvedor irá criar suas querys com mais cuidado em relação a performance e não só focado no resultado.
 
**Speaker(s):**
- Reginaldo Silva
 
*Track and Room*: Database Development - Sala
 
----------------------------------------------------------------------------------- 
 
 
## Title: Obtenha maior produtividade: desenvolvimento e administracao do seu SQL Server
 
**Abstract**:
Monitoring: Top SQLs Statements consumption detection, identifying fragmented indexes
Analyzing: Locks and Blocks detections and actions to avoid this
Rewriting: SQL Codes - changing executing plan
Migrations:  database comparison, schema comparison and data comparison
Analysis services: managing connections MDX and other info
 
**Speaker(s):**
- Gustavo Carneiro
 
*Track and Room*: Database Administration - Auditorio
 
----------------------------------------------------------------------------------- 
 
 
## Title: Por Detrás dos Relatórios
 
**Abstract**:
Nesta sessão você irá descobrir o deve existir por detrás dos relatórios empresariais para que o suporte a tomada de decisão seja rápido, seguro e correto. Após entender os passos que existem até a criação dos reports será possível entender toda a arquitetura de um ambiente de Data Warehouse corporativo.
 
**Speaker(s):**
- Arthur Luz
 
*Track and Room*: BI  Big Data - Anfiteatro
 
----------------------------------------------------------------------------------- 
 
 
## Title: Iniciando com Entity Framework
 
**Abstract**:
Vamos conhecer o EF, entendendo o que ele é, como funciona e como começar a utilizá-lo nos projetos. Teremos exemplos práticos de utilização do mesmo, criando os modelos a partir da importação de um banco de dados existente e criando um novo banco de dados a partir da definição do modelo codificado utilizando Code First e Migrations.
 
**Speaker(s):**
- Aline Favretto
 
*Track and Room*: Database Development - Sala
 
----------------------------------------------------------------------------------- 
 
 
## Title: SQL In-Memory aka Hekaton 101
 
**Abstract**:
Sabe o que é In-Memory – também conhecido como Hekaton? Já leu alguma coisa a respeito de como funciona e/ou onde pode ser interessante utilizá-lo? Nesta sessão, faremos um overview do que é esse recurso, que foi introduzido no SQL 14 e melhorado a cada nova versão, abordando seus princípios básicos, sua arquitetura e alguns dos cenários onde sua implantação pode ser extremamente útil.
 
**Speaker(s):**
- Felipe de Assis
 
*Track and Room*: Database Development - Sala

## Title: Analisando o Plano de Execução através da Query Store
 
**Abstract**:
Através dessa nova feature,  vamos fazer uma análise de um plano de execução que esteja apresentando algum problema de desempenho,  identificando  qual melhor plano usar, para tornar as consultas mais rápidas e performáticas.
 
**Speaker(s):**
- Sulamita Dantas
 
*Track and Room*: Database Development - Sala
 
----------------------------------------------------------------------------------- 
 
 
## Title: Introdução do Azure SQL DataWarehouse
 
**Abstract**:
O SQL Data Warehouse do Azure é um banco de dados baseado em nuvem e expansível com capacidade de processar volumes imensos de dados, relacionais e não relacionais. Criado em nossa arquitetura MPP (processamento paralelo maciço), o SQL Data Warehouse pode lidar com sua carga de trabalho corporativa.
 
**Speaker(s):**
- Vitor Fava
 
*Track and Room*: BI  Big Data - Anfiteatro
 
----------------------------------------------------------------------------------- 
 
 
## Title: Azure SQL Databases
 
**Abstract**:
Conheça mais sobre o Azure SQL Databases e descubra como ele pode ajudar sua empresa na redução de custos e em escalabilidade.
Foque mais no desenvolvimento da sua aplicação e menos na administração dos recursos.
 
**Speaker(s):**
- Flávio Farias
 
*Track and Room*: Database Administration - Auditorio
 
----------------------------------------------------------------------------------- 
 
 
## Title: High Availability and Disaster Recovery Salvando Vidas
 
**Abstract**:
A sessão High Availability and Disaster Recovery salvando vidas,  será um bate papo descontraído, sobre Alta Disponibilidade e Disaster Recovery, passando sobre casos reais de desastres onde o DR literalmente salvou vidas.

Também será abordados exemplos reais de aplicabilidade de HA e DR com o SQL Server 2016, com topologias reais.
 
**Speaker(s):**
- Luiz Henrique Garetti
 
*Track and Room*: Database Administration - Auditorio
 
----------------------------------------------------------------------------------- 
 
 
## Title: Planejamento Estratégico com BI  BIG Data
 
**Abstract**:
Não vamos falar apenas de ferramentas, vamos discutir estratégias de negócio e  avaliar qual ferramenta pode nos apoiar em cada situação. Discussão de estudos de caso, demonstração de ferramentas (incluindo Análise de Vínculos) e, tirar você da zona de conforto são os objetivos dessa sessão! Os alunos que participarem dessa sessão também receberão acesso a um conteúdo exclusivo no formato online.
 
**Speaker(s):**
- Viviane Ribeiro
 
*Track and Room*: BI  Big Data - Anfiteatro
